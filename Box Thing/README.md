# Box Thing

## Description

This is a box with a screen rotary encode and a unnessaryly large switch to control things.
I made my first pcb for this project and it was a lot of fun.

## Insides

- ESP8266
- 1.4" OLED screen
- Rotary encoder
- Big switch
- 1S Lipo battery
